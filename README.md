# 🌾 PaddyPro – Smart Amount Calculator

## 📝 Description

PaddyPro is a simple and user-friendly web application designed to help farmers calculate the final amount they receive after selling paddy. It performs accurate calculations for reduction, final weight, price per kilogram, total earnings, and yield level, making farm calculations quick and hassle-free.

---

## 🚀 Features

* Calculate total reduction (bags × reduction per bag)
* Calculate final weight
* Automatic price per kg calculation
* Total amount calculation
* Dynamic Putti conversion (default 1 Putti = 856 kg)
* Yield analysis (Low / Medium / High)
* Clean and responsive UI
* Print receipt feature

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## ▶️ How to Use

1. Enter Total Net Weight (kg)
2. Enter Number of Bags
3. Enter Reduction per Bag (kg)
4. Enter Total Rate per Putti
5. Click on **Calculate**
6. View the detailed receipt
7. Use **Print Receipt** if needed
